# banqed MVP

banqed is a wardrobe and resale management web app designed to help users understand their wardrobe, track items they may wish to sell, and support more circular consumption habits.

## Current MVP Scope

- Add Item flow
- Wardrobe view
- Sales view
- Shared item data model
- Local storage persistence

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub
- Visual Studio Code

## Project Structure

- `index.html` — app structure
- `style.css` — visual styling
- `script.js` — application logic
- `docs/` — product and technical specifications
- `assets/` — static assets

## Development Priorities

1. App shell and navigation
2. Add Item flow
3. Wardrobe rendering
4. Sales rendering
5. Filtering and sorting
6. Inline editing
