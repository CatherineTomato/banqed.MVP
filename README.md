banqed MVP
Project Overview

banqed MVP is a single-page web application designed to help users catalogue their wardrobe, understand the value of their clothing, and identify items that could be resold.

The project was developed as part of a JavaScript assignment to demonstrate the use of HTML, CSS, and JavaScript in building an interactive web application.

The application focuses on structured item input, dynamic data display, and user interaction through filtering and sorting.

Features
Add Item (Interactive Form)
Guided multi-section form for adding clothing items
Required field validation using JavaScript
Custom multi-select dropdowns (e.g. colour, detail, style)
Dependent dropdown logic (category → item type)
User feedback on submission
Items saved using localStorage
Wardrobe View
Displays all wardrobe items in a structured table
Real-time item count and total value metrics
Dynamic rendering of stored data
Filtering and Sorting
Filter items by multiple attributes (e.g. category, colour, brand)
Sort items by value, date, category, and emotional rating
Active filter/sort tokens that can be removed interactively
Navigation
Single-page navigation system controlled by JavaScript
Sections update dynamically without page reload

How to Run Locally
Download or clone the repository
Open the project folder

JavaScript Interactivity

This project uses JavaScript to add functionality and interactivity, including:

Processing a form submission (Add Item form)
Validating user input and providing feedback
Dynamically updating the DOM to:
render wardrobe items
apply filters and sorting
update metrics
Handling user interactions through event listeners

This meets the assignment requirement for:

at least one form processed by JavaScript
multiple DOM manipulation interactions
interactive user interface elements
